# GSM Arena Scraper

Script ini menggunakan cURL untuk mengambil data HTML dari halaman utama, dan file_get_contents untuk mengambil data HTML dari halaman spesifik setiap smartphone.

# Steps (TERMUX):
- pkg install php
- pkg install apache2
- pkg install nano
- nano scrape.php
- php -S 127.0.0.1:8080
